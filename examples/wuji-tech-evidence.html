舞肌科技：Wuji Hand 2 技术与商业化线索

最后更新：2026-06-22 对象：Wuji Hand / Hand 2 / Wuji Glove

一眼结论

产品判断

舞肌最新产品不是一只孤立的机械手

Wuji Hand 2 的重点是把手接入遥操作、重定向、仿真、ROS2、SDK、Studio、VLA 训练和真实机器人部署链路。创业者要看的不是“它有 20 个自由度”，而是它把手、手套、软件、数据和客户工程连成了一个平台。

技术迭代

从第一代手到 Hand 2，迭代重点在闭环和平台化

第一代 Wuji Hand 已经公开 20 DOF、FOC、约 580g、15N 指尖力、10kg 静态抓握等指标。Hand 2 进一步强调 Ethernet、1000Hz x 20 轴、MIT 力位混合控制、可回驱直驱关节、多轴力/力矩指尖传感和模块化。

商业化缺口

公开资料未见成交价、订单和毛利

目前没有可靠的成交价、配件价、软件订阅费、客户订单、出货量、收入、毛利、售后成本和回款周期。商业判断需要继续找报价单、客户访谈、融资文件或真实采购记录。

产品为什么这样迭代

从执行器、整手、手套、Hand 2 到开源和论文，舞肌的产品迭代指向同一个问题：让灵巧手从硬件样机进入可示教、可仿真、可部署的机器人操作系统。

1. 执行器能力

舞肌早期公开叙事多次指向高扭矩密度电机。对灵巧手来说，这不是背景故事，而是产品可行性的根：重量、指尖力、寿命、体积和控制都受执行器约束。

2. 第一代手

Wuji Hand 官方页给出 20 个主动自由度、串联直驱旋转关节、FOC 矢量控制、15N 指尖力、10kg 整手静态抓握、约 580g 裸手重量、80cm 跌落测试和 30 万次空载循环寿命。这一步证明它不是概念外壳，而是能交付的末端执行硬件。

3. 数据入口

Wuji Glove 的关键不是“配件”，而是示教和数据采集入口。官方页给出 24 x 32 触觉矩阵、526 个有效触觉点、120 FPS、5 个电磁接收模块、6DOF 跟踪、IMU 和以太网通信，这说明舞肌在补机器人手最难规模化的数据问题。

4. Hand 2 平台化

Hand 2 页面强调 1000Hz 控制、Ethernet、MIT 力位混合模式、ROS2、MuJoCo / Isaac Sim、Wuji SDK / Studio、Vision Pro / RealSense / Wuji Glove 遥操作。这一步是把硬件变成开发者和机器人团队能接入的系统。

5. 开源和论文

GitHub 上的 retargeting、ROS2、SDK、mjlab、openpi 仓库，以及 TopoRetarget、Kilohertz-Safe 等论文，说明舞肌正在用真实任务、仿真到实机、VLA 微调、遥操作和安全重定向建立技术可信度。

开源仓库说明的真实方向

截至本页整理时，舞肌 GitHub 组织公开仓库的最新更新集中在 2026 年 6 月前后。它不像普通硬件公司只放说明书，而是在把客户开发链路拆成 SDK、Studio、ROS2、重定向、仿真、VLA 训练和硬件描述文件。

SDK / 工具

wuji-sdk 与 wuji-studio

SDK 说明自动设备发现、连接管理、实时数据流、手部追踪、触觉/电磁传感和 MCAP 记录。Studio 是设备管理、数据可视化和固件升级桌面应用。判断：舞肌在降低客户接入门槛。

重定向

wuji-retargeting

仓库描述支持 Wuji Glove、Vision Pro、RealSense、ZED、视频等输入，WH120 对应网络化 Wuji Hand 2。判断：舞肌把“人怎么教手”作为产品核心，而不是只卖本体。

机器人接入

wujihandros2 与 wuji-hand-teleop

ROS2 驱动强调 1000Hz joint state publishing、实时控制、多手配置和 RViz。Teleop 仓库做实时手和机械臂控制。判断：目标客户很可能是机器人实验室、人形机器人团队和具身智能团队。

仿真 / VLA

wuji-mjlab 与 wuji-openpi

mjlab 做 Wuji Hand 立方体手内重定向 RL 和 sim2real；openpi 面向双臂加 Wuji Hand 的 pi0 / pi0.5 VLA 微调和 ROS2 部署。判断：最新方向已经进入“数据加模型加实机”的完整闭环。

硬件文件

wuji-hand-description 与 hardware-design

URDF、MJCF、mesh、RViz 配置，以及 STEP 格式的转接件和结构文件，能让客户更快做仿真、装配和集成。它们不是拆机报告，但比普通宣传图更接近工程证据。

人员线索

公开 GitHub 组织成员

GitHub 公开组织成员页只显示 Siqiao Huang / knightnemo。这个页面能作为技术贡献者线索，但我没有找到公开证据证明其 CTO 身份，因此不把他写成 CTO。

论文能证明什么

2026-06

TopoRetarget: Interaction-Preserving Retargeting for Dexterous Manipulation

论文用交互图、Laplacian 变形、方向一致性、运动学约束和穿透处理做灵巧操作重定向，报告 Pen-Spin 训练成功率提升 40.6 个百分点，并展示到 Wuji Hand 硬件的零样本迁移。它能证明技术验证方向，不能证明商业订单。

2026-03

Kilohertz-Safe: A Scalable Framework for Constrained Dexterous Retargeting

论文把非线性重定向改写为关节微分空间里的凸 QP，并用控制障碍函数处理安全约束；在 Wuji Hand 平台做硬件验证，报告平均延迟 9.05ms，超过 95% 帧满足安全标准。它说明实时安全重定向是舞肌生态里的关键问题。
